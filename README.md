## Local Installation

### Requisites:
- Docker
- Docker-compose

1. Clone the repository to your local machine:

```
git clone https://github.com/KennysSparda/Portfolio_prod.git
```

2. Run the project:

```
cd portfolio_prod
docker-compose up
```
